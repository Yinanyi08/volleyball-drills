# 排球精英教练 (VolleyPro Trainer) 开发说明

**版本**：v1.2.0 (精简稳定版)
**平台**：微信小程序 (原生)
**AppID**：wxc871ee51d486e0f7
**项目定位**：面向排球教练与系统化训练人群的专业训练项目库工具

---

## 1. 项目背景与目标

### 1.1 背景说明

本项目源自一个基于 React 的排球教学网站，在实际使用中发现：

- Web 端在移动端访问存在性能与交互限制；
- 教练端核心需求集中在「快速查找训练方案」而非复杂功能；
- 微信生态在国内教练群体中具备更低的使用门槛。

因此，本项目对原有 Web 应用进行**功能收敛、结构重构与原生化迁移**，形成一个轻量、高性能、可长期维护的微信小程序版本。

### 1.2 核心目标

- 构建一个**训练导向而非内容导向**的排球训练工具；
- 提供**结构化、可复用、专业一致**的训练项目描述；
- 保证在中低端设备上的**流畅加载与稳定渲染**；
- 为后续专业化扩展（视频、教练体系、会员系统）预留接口。

---

## 2. 功能范围说明 (当前版本)

> 本版本在保持“训练项目工具库”定位不变的前提下，新增 **专项体能训练分页**，用于承载排球专项体能内容，为后续线下训练与多方向扩展提供统一入口。

### 2.1 已实现功能

- **技术训练项目库**：
  - 共 93 个高质量排球专项技术训练项目；
  - 每个项目包含：标题、目标、设置、步骤、教练要点、变体；
- **专项体能训练分页 (新增)**：
  - 独立于技术训练的体能训练入口；
  - 面向排球专项所需的力量、速度、灵敏、稳定性等能力；
  - 采用与技术训练一致的数据驱动结构；
- **双层筛选系统**：
  - 技术 / 体能类型维度；
  - 年龄 / 阶段维度；
- **双语展示机制**：
  - 中文为主，英文作为专业术语对照；
- **详情页沉浸式阅读**：
  - 图文比例优化；
  - 支持训练示意图高清查看。

### 2.2 明确不包含的功能 (设计决策)

- ❌ AI 教练 / 自动生成训练内容；
- ❌ 用户系统 / 登录 / 数据云同步；
- ❌ 社交、评论、点赞等社区功能。

---

## 3. 技术栈与整体架构

### 3.1 技术选型

- **前端框架**：微信小程序原生 (WXML / WXSS / JavaScript)
- **渲染模式**：WebView (以兼容性和稳定性优先)
- **数据组织**：本地 CommonJS 模块化数据
- **状态管理**：页面级 `data` + 本地过滤引擎函数

### 3.2 架构设计原则

- 本地优先 (Local First)，避免不必要的网络请求；
- 数据驱动 UI，页面不直接写死训练内容；
- 所有逻辑保持“可读性优先于抽象层级”。

## 5. 数据层设计规范

### 5.1 技术训练与体能训练的数据分层

- **drills.js**: 承载排球专项技术训练 (传、垫、扣、拦、防守等)；
- **conditioning.js (新增)**: 承载排球专项体能训练 (力量、速度、灵敏、核心稳定)；

### 5.2 数据结构约定

每个项目必须包含：

- `id`: 唯一标识；
- `title`: 中文 (英文括号)；
- `description`: 简短描述；
- `category`: 专项能力/动作类型；
- `ageGroup`: 适用年龄段；
- `setup / steps / coachingTips / variations`;
- `image`: 本地示意图路径。

---

## 6. UI / UX 设计语言

- **极简主义**: 移除搜索框，采用双层胶囊筛选器。
- **玻璃拟态**: 顶部粘性页眉，高斯模糊背景。
- **专业色调**: #0052cc (专业蓝), #ea580c (运动橙), #f8fafc (基底灰)。
